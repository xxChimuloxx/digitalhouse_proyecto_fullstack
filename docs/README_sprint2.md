# Sprint 2 - Maquetado HTML y CSS

## Objetivo del sprint

El objetivo del Sprint 2 fue convertir los wireframes definidos en el Sprint 1 en páginas estáticas utilizando HTML y CSS.

En esta etapa no se implementó funcionalidad real, sino una maqueta navegable que permitiera representar visualmente el sitio.

## Páginas maquetadas

- Home.
- Detalle de producto.
- Carrito de compras.
- Registro de usuario.
- Login de usuario.

## Estructura implementada

```txt
views/
├── index.html
├── productDetail.html
├── productCart.html
├── register.html
└── login.html

styles/
└── styles.css
```

## Características principales

- Navegación entre páginas mediante enlaces.
- Diseño responsive básico.
- Header y footer visuales.
- Cards de productos.
- Formulario de registro.
- Formulario de login.
- Carrito visual.
- Errores de formularios simulados.
- Estética gamer/tecnológica.
