# PixelForge Store

## Descripción del proyecto

PixelForge Store es un e-commerce orientado a la venta de productos gamer, accesorios tecnológicos y componentes para setup. El sitio ofrecerá productos como teclados mecánicos, mouses gamer, auriculares, mousepads, monitores, sillas gamer, notebooks, componentes de PC y accesorios para streaming.

La propuesta busca combinar una experiencia visual moderna con una navegación simple, permitiendo que el usuario pueda explorar productos, consultar sus características principales, agregarlos al carrito y avanzar en el proceso de compra de manera clara e intuitiva.

## Público objetivo

El sitio está dirigido principalmente a jóvenes y adultos interesados en el gaming, la tecnología, el streaming, el home office y el armado de setups personalizados. También apunta a estudiantes, trabajadores remotos y usuarios que buscan mejorar su experiencia digital mediante accesorios de calidad.

La oferta se ajustará a este público mediante una estética moderna, categorías claras, productos destacados, descripciones simples, precios visibles y una experiencia de usuario pensada para facilitar la búsqueda y comparación de productos.

## Sitios de referencia

1. Mercado Libre  
Elegido por ser uno de los marketplaces más utilizados en Argentina y Latinoamérica. Sirve como referencia por su estructura de navegación, sistema de búsqueda, categorías, publicaciones de productos y carrito de compras.

2. Compra Gamer  
Elegido por estar directamente relacionado con la temática del proyecto. Es una referencia clara para productos gamer, componentes de PC, armado de equipos y presentación técnica de productos.

3. FullH4rd  
Elegido por su catálogo de hardware, componentes y periféricos. Resulta útil como referencia para organizar categorías de productos tecnológicos y mostrar especificaciones relevantes.

4. Maximus Gaming Hardware  
Elegido por su estética orientada al público gamer y por la forma en que presenta productos destacados, promociones y categorías de hardware.

5. Amazon  
Elegido como referencia internacional por su experiencia de usuario, sistema de recomendaciones, detalle de producto, opiniones de usuarios y estructura general de e-commerce.

## Diseño visual propuesto

El diseño visual del sitio buscará una estética moderna, tecnológica y gamer, utilizando colores oscuros combinados con acentos luminosos.

### Paleta de colores tentativa

- Fondo principal: negro / gris oscuro
- Color primario: violeta eléctrico
- Color secundario: azul neón
- Color de acción: verde o cian
- Texto principal: blanco
- Texto secundario: gris claro

### Tipografías tentativas

- Títulos: Orbitron, Rajdhani o Montserrat
- Texto general: Roboto, Open Sans o Inter
